Atribut Pada HTML

Atribut Merupakan Informasi Tambahan yang Digunakan di dalam tag pembuka. Informasi ini bisa berupa intruksi untuk memberikan efek warna maupun ketebalan dll. Atribut memiliki 2 tahap, dapat di tulis seperti "(name = "value")".

Heading Pada HTML

<h1>, <h2>, <h3>, <h4>, <h5>, <h6>. Perbedanya adalah dari besar kecilnya text. Contoh tag/element <h1> lebih besar ketimbang tag/element <h2>. Penggunanya juga beragam mulai dari berikut

  • <h1> adalah Heading yang digunakan untuk judul dokumen
  • <h2> adalah Heading yang digunakan sebagai sub dari <h1>
  • <h3> adalah Heading yang digunakan sebagai sub dari <h2>

    • Untuk yang lainya sesuaikan juga dengan programer untuk memerikanya sesuai kebutuhan

    <!DOCTYPE html>
    <html>
    <head>
    <title>Page Title</title>
    </head>
    <body>
    <h1>My First Heading>/h1>
    </body>
    </html>

    Tabel Pada HTML

    Table Merupakan struktur yang digunakan untuk menampilkan informasi dalam bentur baris dan kolom. Untuk membuatnya bisa menggunakan <table>, dan tag <tr> untuk membuat baris dan tag <td> untuk membuat kolom.

    Tag <table> merupakan tag pembuka dalam membuata table pada HTML. <tr> dikenal sebagai table row yang merupakan tag yang digunakan untuk membuat baris dalam HTML. Sedangkan <td> dikenal sebagai table data merupakan tag yang digunakan untuk memebuat kolom dalam baris.

    <!DOCTYPE html>
    <html>
    <head>
    <title>Page Title</title>
    </head>
    <body>
    <tr>
    <td> baris 1 / colom 1 </td>
    <td> baris 2 / colom 2 </td>
    <td> baris 3 / colom 3 </td>
    </tr>
    </body>
    </html>
